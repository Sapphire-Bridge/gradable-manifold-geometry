# Size v2 Behavioral Lock-In Runbook

See also `results/manifold_groups_poc/gradable_size_calibration_dataset_note.md` for the short dataset-and-research narrative used as the current reference note.

## Purpose

This is the gated step before raw patching or geometry.

The goal is not to prove a manifold yet. The goal is to test whether the `size` domain remains behaviorally stable under the controls that matter:

```text
natural contexts
neutral Class A/B/C-style contexts
iso-ratio grid
artificial / inverted standards
fictional-semantic neutral standards
counter-natural explicit standards
```

Primary metric:

```text
unique-side corr(ordered_score, log(value / standard))
```

## Gate

Go to confirmatory raw patching only if:

```text
natural r >= 0.40, CI excludes 0
iso_ratio r >= 0.40, CI excludes 0
at least one semantic-control variant passes:
  fictional_semantic r >= 0.40, CI excludes 0
  or counter_natural follows the explicit standard more than the world prior
within-target p is small for at least natural/iso_ratio/semantic-control
```

If bare `neutral` fails but `fictional_semantic` passes, interpret the result as semantically scaffolded calibration, not context-free ratio arithmetic.

If `normality4` or `binary` passes while the adjective readout fails, treat the failure as a readout-prior issue rather than a failure of gradable calibration.

Treat artificial/inverted standards as a stress test:

```text
positive result: strong evidence against pure world-prior explanation
partial result: informative but not blocking
failure: keep as boundary condition
```

No geometry or SAE/CLT group claims should be made until this behavioral gate passes.

An exploratory raw-patching smoke is reasonable before the semantic-control gate fully passes, but it should be reported as exploratory:

## Cross-Domain Matched-Delta Control

This is the next gate for the manifold-candidate claim. It tests whether matched temperature/age activation deltas carry a generic scalar/uncertainty signal that can move size receivers once filtered through the validated size subspace.

Intervention:

```text
h_size_receiver_patched =
  h_size_receiver + alpha * U_size_k U_size_k^T (h_source_donor - h_source_reference)
```

Default implementation:

```text
scripts/make_gradable_cross_domain_delta_rho_controls.py
scripts/patch_gradable_cross_domain_low_rank_control.py
scripts/run_gradable_cross_domain_low_rank_control.sh
```

Design choices:

```text
U_size_k is fit only on size activations, default fictional_semantic_adjective_counts.
Size receivers default to iso_ratio_adjective_counts, matching the primary held-out direction.
Temperature and age source directions are matched to size directions by signed delta in domain-z space.
Domain-z matching avoids treating temperature deltas, age log-ratios, and size log-ratios as the same raw scale.
Minimum-n gate: each source domain must contribute >=20 unique source pairs and >=20 size pairs.
Raw cross-domain activation replacement is deliberately not used; only projected deltas are injected.
```

## Cross-Domain Delta-Basis Transfer

The source-state PCA transfer was weak: temperature transferred only partially and age did not convincingly transfer. This suggests that the shared calibration geometry may live in the contrast space rather than in dominant raw activation-state PCs.

This gate fits PCA bases on oriented source-domain pair deltas:

```text
U_temperature_delta = PCA(h_temperature_high - h_temperature_low)
U_age_delta         = PCA(h_age_high - h_age_low)
```

and evaluates them on size causal deltas:

```text
h_size_receiver_patched =
  h_size_receiver + alpha * U_source_delta_k U_source_delta_k^T (h_size_donor - h_size_receiver)
```

Default implementation:

```text
scripts/patch_gradable_cross_domain_subspace_transfer.py --source_basis_mode deltas
scripts/run_gradable_cross_domain_delta_basis_transfer.sh
```

Design choices:

```text
Source deltas are oriented from the higher scalar coordinate to the lower scalar coordinate.
Source delta SVD is uncentered, so the mean contrast direction is retained.
Source deltas are unit-normalized by default before SVD, so the basis is not dominated by large-norm prompt-family differences.
The size positive control remains the same L20 size_basis_pca.
Norm-matched random controls use the same projected norm as the source delta basis.
```
